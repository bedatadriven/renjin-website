Renjin documentation

Welcome to the documentation for Renjin, the R interpreter for the Java Virtual Machine. There are three types of guides depending on how you use Renjin:

Analysts

Install Renjin locally or in the cloud and use it for high-performance data analysis.

Analyst guide

Developers

Include Renjin in your Java project or call Java classes from Renjin.

Developer guide
